# TOEIC 攻略トレーナー

昇格・キャリアアップのための TOEIC 対策 & ビジネス英語学習アプリ。単一 HTML ファイルで動作し、スマートフォン対応です。

**▶ アプリを使う: https://azp777.github.io/toeic-app/**

## 機能

- **単語カード** — 600 / 730 / 860+ のスコア帯別頻出単語(各30語)。発音再生つき
- **リーディング** — Part 5 文法(30問)/ Part 6 長文穴埋め / Part 7 読解。全問に日本語解説・全訳・「言い換え」ポイントつき。文書ごとに目標時間と所要時間を計測
- **リスニング** — 音声合成による聞き取り問題。0.8 / 1.0 / 1.2 倍速、アメリカ・イギリス・オーストラリア・ニュージーランドのアクセント切替(ミックスモードあり)
- **ビジネスフレーズ** — 会議・メール・交渉・電話・社交・報告の38表現(瞬間英作文形式)
- **翻訳ツール** — 日本語⇔英語の双方向翻訳(MyMemory API)。DeepL / Google 翻訳へのワンタップ連携

学習記録(連続学習日数・累計解答数・正答率)は端末の localStorage に保存されます。

## 技術

- 依存ライブラリなしの単一 HTML(HTML / CSS / Vanilla JS)
- 音声: Web Speech API(アクセントは端末搭載の音声に依存。非搭載の場合は近いアクセントで代用)
- 翻訳: [MyMemory Translation API](https://mymemory.translated.net/)(無料枠のため1日の利用量に上限あり。機密情報は入力しないでください)
